Er zijn verschillende soorten type Selectors, maak voorbeelden van type selectors, class selectors en descendant selectors

De meest bekende selector (voor mij dan) is de type selector.
Hier een voorbeeld : < style>
h1 {
back ground color : red;
}

Voordeel hier van is dat het eenvoudig , via deze selector krijgen alle h1 tags automatisch een rode achtergrond.

De class selector ziet er als volgend uit:
< style >
.zandbak {
color : green;
}
dus als ik nu de tag schrijf bijvoorbeeld < p class="zandbak" > dan is deze tekst groen< / p > deze selector is ideaal als je niet alle tags dezelfde styling wil hebben.

De descendant selector selecteert alle elementen die afstammen van een bepaald element zoals je zit in dit voorbeeld worden alle < p> -elementen binnen < div> -elementen geselecteerd:
div p { background-color: yellow; }
het voordeel hier aan is is dat je meerdere elementen kan koppelen.

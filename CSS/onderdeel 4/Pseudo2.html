Er zijn 14 verschillende pseudo elementen. Je hebt in de voorgaande opdracht reeds 4 kunnen benoemen. Kun je de andere 10 ook benoemen en aangeven wat ze precies doen?

De andere 10 elementen zijn :
  • ::backdrop
  • ::cue
  • ::First-letter
  • ::File-selector-button
  • ::Marker
  • ::part()
  • ::selection
  • ::slotted()
  • ::target-text
1. Het backdrop element wordt gebruikt als je bijvoorbeeld naar fullscreen mode gaat met een video, de backdrop zorgt er voor dat de achtergrond opgevuld wordt.
2. Het Cue element zorgt er voor dat de tekst overlapping in een video gestyled wordt.
3. First letter styled de eerst letter van een element.
4. Dit element staat voor een input knop.
5. Dit element styled de markers is van een list item.
6. Dit element staat voor elk element wat een matching part attribute heeft.
7. Selection element zorgt voor de styling van tekst op je webpage als je het selecteert.
8. Het Slotted element staat voor elk element wat in een slot geplaatst is in de HTML.
9. Dit element staat voor de tekst waar je naar toe scrollt als de browser support scrolls-to-text fragmenten, dan heb je de keuze hoe je de tekst wil highlighten.

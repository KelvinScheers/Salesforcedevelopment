Geef 4 voorbeelden van Pseudo elementen en beschrijf deze

De elementen die ik gekozen heb zijn :
  • ::after
  • ::before
  • ::first-line
  • ::placeholder

1. Het after element laat je content plaatsen na het element wat je gemaakt hebt.
2. Het before element plaatst content voor het element.
3. Het first-line element past styling toe op de eerste zin van het element.
4. Het placeholder element is de tekst in een antwoord balk op bv een form, deze kan je doormiddel van dit element stylen.

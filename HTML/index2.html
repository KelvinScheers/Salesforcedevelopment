1. Bij het inladen van een bestand is het noodzakelijk om de locatie van het bestand aan te geven , dit kan op twee manieren :

  • Absolute Path
  • Relative Path

Wat houden deze 2 paths in ?

Absolute Path is een verwijzing naar url om zoiets op je pagina te krijgen, dit is niet offline beschikbaar

Relative Path kan je vinden op je eigen pc, deze kan je offline gebruiken waardoor je een website af kan maken voordat deze online gaat.

1. Wat is het verschil tussen het internet en het world wide web?


Het WWW is een klein stukje van het internet, internet verbind alles en iedereen met elkaar wereldwijd, denk aan routers , computers , hardware etc.
Het WWW is een grote verzameling van pagina's die je met een muisklik kan vinden.

Wat is het verschil tussen HTML , CSS en Javascript?


HTML is de fundering en de basis van een website.
CSS bepaalt hoe de HTML er uit ziet , dus de opmaak en de indeling van de website
Javascript is een progammeertaal om HTML en CSS dynamisch te maken.

10. Om een tabel te maken heb je de volgende elementen nodig :

  • Table
  • thead
  • tbody
  • tfoot
  • th
  • tr
  • td

Geef voor elk element een definitie :


Table : is de algehele tabel, dit is de opening en de eind tag van de tabel.
thead : thead gebruik je om alle headers te groeperen.
tbody : tbody gebruik je om alle basis content te groeperen
tfoot : tfoot gebruik je voor de content die we als footer content zien
th : dit is een header in rij.
tr : dit is een rij in een tabel.
td : dit is een cel in een tabel.
